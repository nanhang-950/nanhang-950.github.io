简介

Wails 是一个可以使用 Go 和 Web 开发桌面应用的项目。

创建项目，并选择vue模板。

项目结构

在创建项目后，项目中已经有了模板对应的基本结构代码。

  • main.go和app.go：包含Go代码，处理业务逻辑、数据管理和与前端的通信。
  • frontend文件夹：包含前端代码，比如Vue组件。
  • wails.json：Wails项目的配置文件，定义了如何构建和打包应用。
  • build文件夹：用于存放构建后的应用程序和相关资源。

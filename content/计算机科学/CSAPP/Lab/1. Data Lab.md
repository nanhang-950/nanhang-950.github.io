## 1.bitXor（x,y）

> 只使用 非 和 与位运算实现异或操作

```c
~((~x)&(~y)&(~(x&y)))
```

* 思路：

>德摩根定律

德摩根定律是布尔代数中的基本定律，用于简化逻辑表达式。

1.第一个定律：

```c
~(a & b) = (~a) | (~b)
```

也就是说，“A 和 B的非” 等于 “A的非 或 B的非”

2.第二个定律：

```c
~(a | b) = (~a) & (~b)
```

也就是说，"A 或 B 的非" 等于 "A 的非 和 B 的非"。

  ```c
  a^b=
  1.(a|b)&(~a|~b)
  2.~(~a&~b)&~(a&b)
  3.(a&~b)&(~a&b)
  ```

文件操作

查看文件和目录

1. ls 命令

  • -a：显示所有文件，包括隐藏文件（以“.”开头的文件）。
  • -l：长格式显示详细信息，如权限、拥有者、大小和修改时间。
  • -d：显示目录本身的属性，而不是显示目录中的内容。
  • -h：人性化显示文件大小，以 K、M、G 等单位表示文件大小。

2. file 命令

  • 查看文件类型和详细属性，例如文本文件、二进制文件等。

创建文件和目录

1. mkdir 命令

  • -p：递归创建目录，自动创建不存在的父目录。

2. touch 命令

  • 用于创建空文件或更新文件的时间戳。

3. cp 命令

  • -r：复制目录及其内容。
  • -p：保留文件的原始属性，如修改时间和权限。

示例：

删除文件和目录

1. rm 命令

  • -f：强制删除文件，不提示确认。
  • -r：递归删除目录及其内容。

示例：

查找文件

1. locate 命令

  • 快速查找包含指定关键字的文件。

2. find 命令

  • -name：按文件名查找。
  • -type：按文件类型查找（f 为文件，d 为目录）。
  • -size：按文件大小查找。
  • -exec：对查找到的文件执行指定命令。

命令历史

1. history 命令

  • 显示命令历史，按编号显示命令。
  • !<编号>：执行历史中的指定命令。
  • -c：清空历史命令。
  • -w：保存历史到文件。

示例：

通配符和扩展符

  • *：匹配任意数量的字符。
  • ?：匹配单个字符。
  • [ ]：匹配指定范围内的字符。
  • {}：包含逗号分隔的列表，自动展开为多个路径或文件名。

文件内容操作

查看文件内容

1. cat 命令

  • -n：显示文件时加上行号。

2. more 和 less 命令

  • 用于分页查看文件内容。

3. head 和 tail 命令

  • head -n：查看文件前 n 行。
  • tail -n：查看文件后 n 行。
  • tail -f：实时显示文件新增的内容，常用于监控日志文件。

文件统计

1. wc 命令

  • -l：统计行数。
  • -m：统计字符数。
  • -c：统计字节数。

文件搜索

1. grep 命令

  • -n：显示匹配行及行号。
  • -i：忽略大小写。
  • -v：显示不匹配的行。

文件对比

diff 命令

  • 比较两个文件，显示差异。

时间管理

1. date 命令

  • 查看当前系统时间。

2. hwclock 命令

  • -w：将系统时间写入硬件时钟。
  • -s：将硬件时钟的时间设置为系统时间。

其它常用命令

1. which 命令

  • 查找命令的实际路径。

2. stat 命令

  • 查看文件的详细属性，包括修改时间、权限等。

3. ln 命令

  • -s：创建软链接。

查看帮助

1. help 命令

  • 查看 Shell 内置命令的帮助信息。

2. man 命令

  • 查看命令的手册页，获取详细的使用说明。

重定向

标准输入输出

  • <：输入重定向，将文件作为命令的输入。
  • >：输出重定向，将命令的输出保存到文件。
  • &>：将标准输出和错误输出一起重定向到文件。

《操作系统真象还原》chapter1 环境搭建

环境：wsl Ubuntu22.04

安装GCC

安装gcc

apt install gcc

安装NASM

NASM是一个多平台的汇编编译器，语法简洁易用。

apt install nasm

安装Bochs

Bochs是一个用于模拟硬件的虚拟机。

包管理安装

apt install bochs

源码编译安装

  • 下载源代码

wget https://sourceforge.net/projects/bochs/files/bochs/2.7/bochs-2.7.tar.gz/download

  • 编译安装

编译安装的目录下有样本文件：share/doc/bochs/bochsrc-sample.txt

将代码保存为mbr.s文件

执行命令nasm mbr.s -o test

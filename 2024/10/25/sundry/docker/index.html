docker

查看镜像

  • docker image ls
  • docker images

拉取镜像

官方镜像

  • docker image pull 镜像名称
  • 简写：docker pull 镜像名称
    • 例如：docker pull ubuntu

个人镜像

  • docker pull 仓库名称/镜像名称

第三方仓库拉取

  • docker pull 第三方仓库地址/仓库名称/镜像名称

删除镜像

  • docker image rm 镜像名或镜像ID 或 docker rmi 镜像名或镜像ID

加载镜像

  • docker run [可选参数] 镜像名 [传入的命令]

常用参数：

  • -i：以交互模式运行容器。
  • -d：后台运行容器，创建守护式容器。
  • -t：为容器分配一个伪终端。
  • --name：为容器命名（不支持中文字符）。
  • -v：目录映射（宿主机目录:容器目录）。
  • -p：端口映射（宿主机端口:容器端口）。
  • --network=host：使用主机的网络环境。

查看容器

查看正在运行的容器

  • docker ps

查看所有容器

  • docker ps -a

常用过滤器

  • 通过名字：docker ps -f name=指定名字
  • 显示最新容器：docker ps -l
  • 仅显示容器ID：docker ps -q
  • 显示容器大小：docker ps -s

启动和关闭容器

停止容器

  • docker container stop 容器名或ID
  • 简写：docker stop 容器名或ID

强制关闭容器

  • docker container kill 容器名或ID
  • 简写：docker kill 容器名或ID

启动容器

  • docker container start 容器名或ID
  • 简写：docker start 容器名或ID

操作后台容器

执行命令

  • docker exec -it 容器名或ID 命令
    • 例如：docker exec -it kali /bin/bash

附加到容器

  • docker attach 容器名或ID

删除容器

  • docker rm 容器名或ID

制作容器镜像

将容器保存为镜像

  • docker commit 容器名 镜像名

镜像打包备份

  • docker save -o 文件名 镜像名

镜像解压

  • docker load -i 文件路径/备份文件

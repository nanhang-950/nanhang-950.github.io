部署 Dify

Dify 的部署方法有两种，Docker compose 部署或源码部署。

这里就只讲 Docker 部署的内容，这要比源码部署便捷很多。

计算机的硬件要求：

  • CPU >= 2 核
  • RAM >= 4 G

软件要求：

  • 使用 Docker 部署在 Windows 系统下必须已安装 Docker Desktop。

首先克隆 Dify 项目的代码仓库，亦或者直接在 Dify 项目中下载项目压缩包

github：langgenius/dify：Dify 是一个开源的 LLM 应用程序开发平台。Dify 的直观界面结合了 AI 工作流程、RAG 管道、代理功能、模型管理、可观测性功能等，让您可以快速从原型到生产。

点击 “下载 ZIP”
